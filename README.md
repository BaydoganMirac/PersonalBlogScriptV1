## How To Install
1. Create a database
2. Import blogum.sql
3. Edit  "src/config.db.php"
```php
$host               = "yourhostname";
$username           = "yourdbusername";
$password           = "yourdbpassword";
$database           = "yourdbname";
```
4. Open http://yourdomain/admin/ 
5. **Username** : Admin  **Password**: 123
5. Edit Website
